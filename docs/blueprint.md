# **App Name**: VitaliaConnect

## Core Features:

- Real-time Communication: Enables real-time messaging and video calls between families and care centers.
- Resident Monitoring: Allows monitoring of resident activities and well-being through sensor data (simulated for MVP).
- Notification System: Sends real-time notifications for important events, such as medication reminders and appointments.
- Appointment Scheduling: Facilitates scheduling and management of medical appointments and other activities.
- Information Summarizer: Uses AI to summarize resident data to facilitate staff to more quickly onboard patients. Acts as a summarization tool.
- App Screenshots: Displays screenshots of the app's user interface to showcase its features and design.

## Style Guidelines:

- Primary color: Soft light blue (#ADD8E6) to evoke calmness and trust.
- Background color: Very light blue (#F0F8FF), almost white, for a clean and minimal aesthetic.
- Accent color: Pale blue (#B0E2FF) to highlight key elements without being overwhelming.
- Body and headline font: 'Inter' for a modern and clean sans-serif look.
- Use outline-style icons to maintain a modern and minimal aesthetic.
- Centering content and using a clear visual hierarchy to improve readability.
- Subtle fade-in and slide animations for a smooth user experience when scrolling.
